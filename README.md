## 🚀 Overview

FinPilot AI is a modern AI-powered personal finance platform designed to simplify money management while encouraging healthy financial habits. The platform combines intelligent expense tracking, budgeting, savings management, AI-powered financial advisors, predictive insights, and gamification to deliver an engaging and personalized financial experience.

Built using **React, TypeScript, Vite, Express.js, Tailwind CSS, Google Gemini, LangChain, and LangGraph**, FinPilot AI provides a responsive SaaS-style interface with intelligent financial analysis, conversational AI assistance, and interactive visualizations.
